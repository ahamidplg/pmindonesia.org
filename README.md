# Project Management Indonesia (PMI) Website

A high-conversion funnel and corporate profile website designed for **Project Management Indonesia**. This application serves as a professional hub for project management consulting, PMP® certification training, and digital PM system implementation.

## 🚀 Overview

This website is built with a "Results-Driven" philosophy, combining a clean, modern corporate aesthetic with strategic lead generation funnels. It features glassmorphism elements, professional typography, and smooth interactive animations to build trust with high-level professionals and corporate clients.

## ✨ Key Features

- **Strategic Hero Section**: High-impact messaging with clear Call-to-Action (CTA) for training registration and free webinars.
- **Service Catalog**: Detailed breakdown of PMI services including PMP® Training, PMO Consulting, Software Implementation, and Project Audits.
- **Conversion Funnel**: Integrated "Free Webinar" section designed to capture top-of-funnel leads before moving them to paid offerings.
- **Lead Generation**: A functional, animated contact form with success states for capturing consulting inquiries.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices using Tailwind CSS's mobile-first approach.
- **Modern UI/UX**:
  - **Typography**: Inter (Sans) for readability and Space Grotesk (Display) for a technical, modern feel.
  - **Animations**: Subtle entry effects and interactive states powered by `motion`.
  - **Aesthetics**: Professional Brand Blue palette with high-contrast Orange accents for conversion elements.

## 🛠 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠 Installation & Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## 📝 Design Philosophy

The design uses a **Professional Master-Gate** pattern:
1. **Awareness**: Social proof and global standards in the Hero section.
2. **Interest**: Service breakdown with clean grid layouts.
3. **Desire**: Educational value through the Webinar funnel.
4. **Action**: Simplified, non-intimidating lead forms.
